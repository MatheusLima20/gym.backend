Scalable backend API built with Node.js and TypeScript following a modular architecture pattern.

---

## 🚀 Features

- JWT authentication
- User management
- Role-based permissions
- Product management
- Order management
- Cashier management
- File upload
- Logging system

---

## 🛠 Tech Stack

- Node.js
- TypeScript
- Express.js
- MySQL
- Docker
- TypeORM
